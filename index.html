Enable access for assistive devices

Before running the application you will need to enable access for assistive devices. If you do not do this, Plover will not work.

Pre-Mavericks

  1. Open System Preferences
  2. Open "Universal Access"
  3. Check the box next to "Enable access for assistive devices"

Mavericks

  1. Open System Preferences
  2. Click the Security & Privacy icon.
  3. Select the Privacy tab.
  4. Select Accessibility from the list on the left side of the Privacy pane.
  5. Click the lock icon in the lower left corner of the window and enter an admin account and password when prompted.
  6. Drag Plover into area on the right.

Debian/Ubuntu

There is no package yet so an installation requires installing all the dependencies. First you must have python 2.7. Then, run the following commands:

cd
sudo apt-get install python-xlib python-wxgtk2.8 wmctrl python-dev python-pip
sudo pip install -U appdirs simplejson pyserial
wget https://github.com/plover/plover/archive/{{latest_version}}.tar.gz
tar -zxf {{latest_version}}.tar.gz
cd plover-{{latest_version | limitTo:(-latest_version.length + 1)}}
sudo python setup.py install

Once this is done then you should be able to run plover from the applications menu or from the command line with:

/usr/local/bin/plover

Fedora

To install plover along with the dependencies on Fedora 20 and above, run the following commands: 

                sudo yum install -y wxPython pyserial python-lockfile python-serial python-xlib
                sudo pip install plover

Running Plover

Starting the application will bring up a small window with a red 'P' button. At this point, Plover is inactive. To activate Plover, click the red 'P' button. The 'P' will turn green to indicate that Plover is active and ready to translate stenography keystrokes into English text. Clicking the green button will cause it to turn red, which means Plover is again in the inactive state.

If you run into any problems, seek help at the Forum or mailing list.
